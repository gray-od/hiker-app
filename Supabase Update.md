Quick product announcements
Updates
The Postgres log_connections setting now defaults to off for new projects on all tiers starting July 9, and existing Free and Pro projects will be migrated to the new default. [GitHub Discussion]
pg_graphql v1.6.2 is now in production with GraphQL schema introspection off by default, so enable it per schema if you use GraphiQL or codegen tools. [GitHub Discussion]
Audit Log Drains are now available, so you can stream your project’s audit logs to an external destination. [Docs]
Connect now copies every environment variable @supabase/server needs in a single click. [Demo]
Self-hosted Docker defaults changed: API_EXTERNAL_URL now includes the /auth/v1 prefix and the default image moves to Postgres 17. [GitHub Discussion]
